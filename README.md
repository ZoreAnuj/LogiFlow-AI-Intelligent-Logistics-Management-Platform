# LogiFlow AI: Intelligent Logistics Management Platform

LogiFlow AI is an experimental platform that applies multi-step AI agents and vector search to automate and optimize logistics operations. It explores how autonomous AI workers can enhance predictive maintenance, dynamic routing, and emergency response within supply chains.

## Key Features
*   **Multi-Step AI Agents:** Orchestrates sequential reasoning tasks for complex logistics decision-making.
*   **Vector-Powered Search:** Utilizes TiDB Serverless for semantic search across logistics data and documentation.
*   **Predictive Analytics:** Models for forecasting maintenance needs and optimizing delivery routes.
*   **Automated Emergency Response:** Triggers and manages AI-driven protocols for supply chain disruptions.

## Tech Stack
TiDB Serverless, Python, AI Agent Frameworks, Vector Search APIs

## Getting Started
1.  Clone the repo: `git clone https://github.com/zoreanuj/LogiFlow-AI-Intelligent-Logistics-Management-Platform.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Configure your environment variables in `.env`.
4.  Run the main agent script: `python src/main_agent.py`
