# LogiFlow AI: Intelligent Logistics Management Platform

LogiFlow AI is an intelligent logistics platform that leverages multi-step AI agents and vector search to automate and optimize supply chain operations. It explores how AI can enhance decision-making in logistics through predictive analytics and autonomous workflows.

## Key Features
- Predictive maintenance alerts using anomaly detection on sensor data
- Dynamic route optimization with real-time traffic and constraint analysis
- AI-powered emergency response coordination for supply chain disruptions
- Unified logistics data querying via TiDB Serverless vector search

## Tech Stack
- **Backend:** Python, FastAPI
- **AI/ML:** LangChain, OpenAI API, scikit-learn
- **Database:** TiDB Serverless (vector search)
- **Infrastructure:** Docker, GitHub Actions

## Getting Started
```bash
git clone https://github.com/zoreanuj/LogiFlow-AI-Intelligent-Logistics-Management-Platform.git
cd LogiFlow-AI-Intelligent-Logistics-Management-Platform
pip install -r requirements.txt
python app/main.py
```
